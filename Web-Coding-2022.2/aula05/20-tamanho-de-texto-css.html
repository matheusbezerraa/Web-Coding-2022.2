Tamanho de texto

As medidas de tamanho de texto podem ser representadas por:

  • px: tamanho fixo
  • %: tamanho relativo
  • em: tamanho relativo em relação ao container pai

Exemplo

font-size 16px: é o valor padrão quando nenhum tamanho de fonte é declarado.

font-size 100%: compatível ao tamanho padrão de 16px. Se o font-size for declarado como 200% então a fonte terá duas vezes o tamanho padrão, ou seja, o equivalente a 32px.

Caso uma div seja estilizada com uma fonte de por exemplo 60px e outro trecho deste container seja estilizado com 0.5em, então a fonte terá 30px, caso seja 2em, a fonte terá 120px, ou seja, em é um valor relativo.
