Elemento inline

Sua largura ocupa apenas o tamanho do conteúdo elemento. Ficam um ao lado do outro
Exemplo:
    <a>, <span> e <img>

ELEMENTO INLINE - SPAN


Podemos transformar um elemento de comportamento em inline para block usando:
display: block

ELEMENTO INLINE -> BLOCK (SPAN)

Elemento block

Sua largura ocupa todo o espaçamento da tela e ficam um abaixo do outro.
Exemplo:
    <h1>, <p> e <table>

ELEMENTO BLOCK- H1


Podemos transformar um elemento de comportamento em block para inline usando:
display: inline

ELEMENTO BLOCK -> INLINE (H1)

Elemento inline-block

Comportamento híbrido, sua largura ocupa apenas o tamanho do conteúdo elemento, mas ficam um abaixo do outro.
Exemplo:
    <display: inline-block>
