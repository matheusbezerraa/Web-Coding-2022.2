Diferente de outras tags como h1, p, e outras, os links possuem estados:
  • Links não visitados: a:link
  • Links visitados: a:visited
  • Links "hover": a:hover
  • Links ativos: a:active

Para formatar um link não visitado utilizamos o seletor:

a:link{
     Formatação
}

Para formatar um link já visitado utilizammos o seletor:

a:visited{
     Formatação
}

Para formatar um link em que o cursor do mouse está sobre ele (hover) utilizamos o seletor:

a:hover{
     Formatação
}

Para formatar um link ativo (clicado e segurado) utilizamos o seletor:

a:active{
     Formatação
}
