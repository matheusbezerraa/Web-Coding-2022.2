Modelo de Caixa - Box Model

Elementos HTML podem ser considerados caixas que contém os seguintes atributos:

Onde:

  • Conteúdo: é o conteúdo da caixa, onde textos e imagens aparecemm.
  • Padding (top-right-bottom-left): como "preenchimento", é a área ao redor do conteúdo. É transparente.
  • Border (top-right-bottom-left): ou "borda", é o que limita (está ao redor) do padding e do conteúdo.
  • Margin (top-right-bottom-left): ou "margem", é a área ao redor (por fora) da borda. É transparente.
