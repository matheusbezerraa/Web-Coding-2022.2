External CSS

Ou "estilo externo", neste caso o CSS será criado em um arquivo externoo (exemplo: estilo.css) e todo o código de estilo será escrito neste arquivo.

O arquivo será importado nas páginas em que os estilos serão aplicados. Para isso deve-se fazer referência ao arquivo no head do documento:

<head>
    <title>Título</title>
    <meta charset = "UTF-8">
    <link ref = "stylesheet" href = "caminho">
</head>

Para exemplo, clique aqui e inspecione a página (Ctrl+Shift+i).
