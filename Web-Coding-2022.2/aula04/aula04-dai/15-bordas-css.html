Bordas

Propriedades CSS de bordas permitem a aplicação de estilos, largura e cor em um elemento de borda usando a seguinte estrutura:

atributo: largura estilo cor;

Exemplo

border: 1px solid red;

Estilos de bordas

  • Dotted
  • Dashed
  • Solid
  • Double
  • Groove
  • Ridge
  • Inset
  • Outset
  • None
  • Hidden

Um elemento pode ter até quatro valores para estilo de borda (top - right - bottom - left)
