Internal CSS

Ou "estilo interno", deverá ser escrito dentro do head usandoa tag <style></style> .

Os estilos são formatados através de seletores que possuem a seguinte estrutura:

seletor{
    atributo:valor;
}

Para ver um exemplo, clique aqui e inspecione a página (Ctrl+Shift+i).

Neste exemplo, todas as tags de parágrado ficarão vermelhas e as tags de cabeçalho azuis.
