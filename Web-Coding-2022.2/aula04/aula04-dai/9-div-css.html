DIV

É um elemento de divisão, a tag <div></div> funciona como um container que agrupa elementos paar uso de estilos (classes e ids) que compartilham valores e atributos.

uma div ocupa todo o espaçamento da tela, pois é um elemento block

Possui a seguinte estrutura:

<div>
    Conteúdo <p>, <h1>, etc.
</div><

Para ver um exemplo, clique aqui e inspecione a página (Ctrl+Shift+i).
