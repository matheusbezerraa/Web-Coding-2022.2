Classes

Supondo que temos 10 parágrados na nossa página, mas só queremos três deles na cor vermelha. Vamos criar uma classe chamada vermelho a aplicar somente aos parágrafos desejados

Para criar uma classe no Internal CSS usamos a seguinte estrutura:

<style>

    .nomeDaClasse{
        atributo1:valor;
        atributo2:valor;
    }

</style>

Para aplicar a classe a uma tag desejada usamos a seguinte estrutura:

<nomeDaTag class="nomeDaClasse"></nomeDaTag>

Para ver um exemplo, clique aqui e inspecione a página (Ctrl+Shift+i).
