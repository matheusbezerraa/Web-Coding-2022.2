IDs

São usados apenas em um único elemento por página, para declarar um ID usa-se a seguinte estrutura:

#nomeDoId{
    atributo1:valor;
    atributo2:valor;
}

Para aplicar o ID a uma tag desejada usamos a seguinte estrutura:

<nomeDoId id="nomeDaClasse"></nomeDoId>

É mais utilizado para estruturar páginas

Exemplo

Queremos que a cor de fundo (background) de nossa página seja cinza claro. Cor de fundo é um elemento que só tem um por página.
