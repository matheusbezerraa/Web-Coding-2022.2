Cascading Style Sheet - CSS

Introdução

Também chamado de Folha de Estilo em Cascata, o CSS é utilizado para estilizar elementos escritos em uma linguagem de marcação (Markup Language) como o HTML, separando sua representação visual do site.

HTML não foi projetado para ter tags de formatação, deste modo a relação do CSS com o HTML é bastante forte, sendo o HTML a base de um site e o CSS trata de todo o seu estilo (estética).

Vantagens de usar CSS

Dentre as vantagens de usar CSS associado ao HMTL podemos citar:

  • Economia de tempo de criação e manutenção do site, uma vez que o código de formatação fica isolado, na maioria das vezes em um único arquivo CSS.
  • Redução do código HTML.
  • Navegadores podem carregar as páginas até 70% mais rápido.
  • Criação de designs sofisticados.
  • Entre outros.

Formas de inserir CSS nas páginas HTML

Existem três formas de inserir uma folha de estilo em uma página HMTL.

  • Inline CSS ou CSS inline
  • Internal CSS ou CSS interno
  • External CSS ou CSS externo
